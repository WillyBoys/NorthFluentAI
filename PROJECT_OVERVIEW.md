# PROJECT_OVERVIEW.md

## Overview

NorthFluent AI is a beginner friendly AI education platform. The goal is to teach people what AI is, where it helps, where it fails, and how to use it effectively and responsibly.

This project should begin as a small, polished educational website with a simple interactive tool. It should be structured so it can grow into a larger platform later.

## Primary Goal of the MVP

Deliver a clean and useful first version of the platform that includes:
- A clear homepage
- Educational content pages for beginner AI literacy
- A Prompt Improver Tool
- A layout and content structure that can scale later

## Target Audience

The first version should be written for beginners, especially:
- Students
- Job seekers
- Professionals
- Curious learners

Assume the audience has little or no prior AI knowledge.

## Product Promise

NorthFluent AI should help beginners become fluent in AI, not dependent on it.

This means the site should:
- Explain concepts clearly
- Show practical value
- Teach users to think critically
- Emphasize human oversight
- Avoid exaggerated claims about AI

## Core Questions the Site Should Answer

1. What is AI?
2. How does AI work at a basic level?
3. What can AI help me do?
4. What are the weaknesses of AI?
5. How do I write better prompts?
6. How do I use AI responsibly?

## MVP Sitemap

The site should include these core pages:

1. Home
2. AI Basics
3. Prompt Better
4. Strengths and Limits
5. About
6. Prompt Improver Tool

Optional additions later:
- Beginner Quiz
- AI tools recommendation page
- Real world examples section

## Navigation Structure

Recommended main navigation:
- Home
- AI Basics
- Prompt Better
- Strengths and Limits
- About
- Try the Tool

## Page Goals

### Home
Introduce the project and guide users to start learning.

### AI Basics
Explain AI concepts in a very approachable way.

### Prompt Better
Teach better prompting through structure and examples.

### Strengths and Limits
Show what AI does well and where it needs human review.

### About
Share mission, audience, and vision.

### Prompt Improver Tool
Provide a simple interactive experience that makes the platform feel practical and useful.

## Core Learning Philosophy

The educational approach should be:
- Beginner first
- Practical
- Honest
- Balanced
- Encouraging

NorthFluent AI should not present AI as magic.
It should present AI as a tool that can be very useful when used with judgment.

## Recommended User Experience

A user should be able to:
- Land on the homepage and quickly understand the purpose of the site
- Move through the educational pages in a logical order
- Try the Prompt Improver Tool with little friction
- Leave with a clearer and more realistic understanding of AI

## Recommended User Flow

### Main path
Home → AI Basics → Prompt Better → Prompt Improver Tool

### Supporting path
Home → Strengths and Limits → About

## Suggested Tech Direction

Recommended stack:
- React
- Vite
- TypeScript

Recommended style direction:
- Clean layout
- Responsive design
- Reusable components
- Content stored in easy to edit data structures when possible

## Guardrails

Do not add:
- Authentication
- Payments
- Dashboards
- Complex APIs
- Database driven CMS features
- Feature creep that weakens the MVP

## Success Criteria for the First Version

The MVP is successful if it:
- Looks polished
- Explains the topic clearly
- Feels beginner friendly
- Includes a functional Prompt Improver Tool
- Has a structure that is easy to expand later
